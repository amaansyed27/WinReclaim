01 / THE PROBLEM

Storage pressure is rarely where you first look.

Developer tools, local models, emulators, package caches and build output can quietly consume tens of gigabytes. Generic cleaners show a size. WinReclaim adds context: ownership, recoverability, confidence and the cost of getting that data back.

02 / STORAGE INTELLIGENCE

More than a disk scanner.

WinReclaim turns raw folder sizes into an explainable cleanup workflow.

01

Storage Time Machine

Each completed scan creates a local snapshot. Compare scans to see what grew, what shrank, which workload probably caused it, and whether a verified cleanup action exists.

02

Reclaim Passports

Every finding includes its likely owner, recovery class, estimated recovery time, confidence score and evidence used by the classifier.

03

Reclaim Simulation

Preview projected free space, estimated reclaim, recovery categories, protected items and likely recovery time before executing an immutable cleanup plan.

04

Safe Undo Vault

Eligible temp files and crash dumps move into a compressed local vault with seven-day retention, manifest-backed restore and no destination overwrites.

05

Quick to Ultra scan profiles

Start with common cleanup locations or inspect project output, AppData, unknown large directories and verified Windows-drive caches. Protected roots and reparse points remain excluded.

03 / SAFETY BY CONSTRUCTION

Cleanup cannot outrun the rules.

The frontend never receives permission to delete arbitrary paths. Rust creates a hashed, immutable plan from allowlisted findings and compiled cleanup adapters. Unknown discoveries remain inspection-only.

Read the safety model →

06 / PRIVACY

Your storage map belongs on your computer.

Scans, dynamic discoveries, snapshots, passports, plans, cleanup records, vault data and receipts are processed locally. WinReclaim includes no telemetry.

  • File paths and directory trees are not uploaded
  • Project names and receipts remain on-device
  • Update checks contact only GitHub Releases
  • The optional Storage Assistant runs locally
